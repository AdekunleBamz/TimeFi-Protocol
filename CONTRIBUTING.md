# Contributing to TimeFi Protocol

Thank you for your interest in contributing!

## Development Setup

### Prerequisites
- Node.js 18+
- Clarinet CLI
- Git

### Getting Started

1. Fork the repository
2. Clone: `git clone https://github.com/YOUR-USERNAME/TimeFi-Protocol.git`
3. Install: `npm install`
4. Test: `npm test`

## Branch Naming
- `feat/` - New features
- `fix/` - Bug fixes
- `docs/` - Documentation
- `test/` - Test additions

## Commit Messages
- `feat:` - New feature
- `fix:` - Bug fix
- `docs:` - Documentation
- `test:` - Tests

## Running Tests
```
clarinet check
clarinet test
```

## Code Style
- Clear, descriptive names
- Comments for complex logic
- Keep functions focused

## Questions?
Open an issue for discussion.
